# Kotlin Contact Manager

This is a console-based Contact Manager application built using Kotlin.

## Features
- Add new contacts
- View all contacts
- Filter favorite contacts
- Filter contacts with email

## Concepts Used
- Data Classes
- Null Handling
- Higher-Order Functions

## How to Run
1. Compile the code:
   kotlinc Main.kt -include-runtime -d main.jar

2. Run the program:
   java -jar main.jar
